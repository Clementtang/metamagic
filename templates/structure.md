# 結構化優化模板

## 目標

改善 prompt 的組織結構，使其更易於理解和遵循。

## 檢查清單

- [ ] 使用清晰的分段
- [ ] 重要資訊放在前面
- [ ] 使用列表呈現多個要點
- [ ] 使用標題區分不同區塊
- [ ] 指示有明確的順序

## 建議結構

```markdown
# 角色/背景

[定義 AI 的角色和任務背景]

# 任務

[主要任務描述]

# 要求

- 要求 1
- 要求 2
- 要求 3

# 限制

- 不要 X
- 避免 Y

# 輸出格式

[期望的輸出格式]

# 範例（選用）

[輸入/輸出範例]
```

## 結構化技巧

| 技巧         | 說明                 |
| ------------ | -------------------- |
| **倒金字塔** | 最重要的資訊放最前面 |
| **編號列表** | 有順序的步驟用編號   |
| **項目符號** | 無順序的要點用符號   |
| **分隔線**   | 區分不同區塊         |
| **粗體**     | 強調關鍵詞           |

## 優化步驟

1. 分析原始 prompt 的結構問題
2. 將內容分類到適當的區塊
3. 調整順序，重要資訊優先
4. 使用 Markdown 格式增加可讀性
