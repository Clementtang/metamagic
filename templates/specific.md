# 具體化優化模板

## 目標

增加具體細節和限制條件，讓 AI 更準確理解需求。

## 檢查清單

- [ ] 定義輸出格式（長度、結構、語言）
- [ ] 指定目標受眾
- [ ] 設定邊界條件（什麼不要做）
- [ ] 提供背景脈絡
- [ ] 明確成功標準

## 具體化維度

| 維度     | 問題               | 範例                      |
| -------- | ------------------ | ------------------------- |
| **格式** | 輸出應該長什麼樣？ | JSON、Markdown、純文字    |
| **長度** | 需要多詳細？       | 100字、3段、5點列表       |
| **語調** | 用什麼風格？       | 專業、友善、幽默          |
| **受眾** | 給誰看？           | 工程師、新手、主管        |
| **限制** | 什麼不要？         | 不要用術語、不要超過500字 |

## 優化步驟

1. 分析原始 prompt 缺少哪些具體資訊
2. 補充格式、長度、語調等要求
3. 加入必要的背景脈絡
4. 設定明確的限制條件
