# 清晰度優化模板

## 目標

讓 prompt 更清晰易懂，消除歧義。

## 檢查清單

- [ ] 移除模糊用詞（如：好的、適當的、一些）
- [ ] 將抽象概念轉為具體描述
- [ ] 確保每個句子只表達一個意思
- [ ] 使用主動語態
- [ ] 避免雙重否定

## 常見問題與修正

| 模糊表達       | 改進方式                          |
| -------------- | --------------------------------- |
| 「寫得好一點」 | 「使用簡潔句子，每段不超過 3 句」 |
| 「適當的格式」 | 「使用 Markdown，包含標題和列表」 |
| 「一些例子」   | 「提供 3 個具體範例」             |
| 「儘快完成」   | 「在 100 字以內回答」             |

## 優化步驟

1. 找出所有模糊或抽象的詞彙
2. 為每個模糊詞提供具體定義或數值
3. 重新組織句子，確保邏輯流暢
4. 檢查是否有多重解釋的可能
