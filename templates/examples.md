# 範例補充優化模板

## 目標

透過輸入/輸出範例，讓 AI 更準確理解期望。

## 檢查清單

- [ ] 提供至少一個完整的輸入/輸出範例
- [ ] 範例涵蓋主要使用情境
- [ ] 範例展示期望的格式和風格
- [ ] 如果有邊界情況，也提供範例

## 範例類型

| 類型         | 用途             | 數量建議 |
| ------------ | ---------------- | -------- |
| **正面範例** | 展示期望的輸出   | 1-2 個   |
| **負面範例** | 展示不要的輸出   | 0-1 個   |
| **邊界範例** | 展示特殊情況處理 | 視需要   |

## 範例格式建議

```
## 範例

輸入：[輸入內容]

輸出：[期望的輸出]

---

輸入：[另一個輸入]

輸出：[對應的輸出]
```

## 優化步驟

1. 判斷是否需要範例（複雜任務通常需要）
2. 設計 1-2 個代表性的輸入
3. 撰寫對應的期望輸出
4. 確保範例與指示一致
